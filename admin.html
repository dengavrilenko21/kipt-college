Админ-панель

Редактирование контента сайта · изменения коммитятся прямо в GitHub

Как получить токен (один раз):
  1. GitHub → аватар → Settings → внизу слева Developer settings
  2. Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access: Only select repositories → выбери kipt-college
  4. Permissions → Repository permissions → Contents: Read and write
  5. Generate token → скопируй и вставь сюда
Токен хранится только в твоём браузере и никуда, кроме GitHub API, не отправляется. Не публикуй его и не пересылай никому.
